# AccessLint

*[English version](README.md)*

CLI без зависимостей на Node.js, который сканирует HTML-файлы на типовые проблемы доступности по WCAG 2.1 — отсутствующие атрибуты `alt`, неподписанные поля форм, неописательный текст ссылок, отсутствующий атрибут `lang` и пропущенные уровни заголовков.

## Зачем

Миллионы сайтов недоступны людям, использующим скринчитатели и другие вспомогательные технологии. Полный аудит доступности дорог — AccessLint даёт разработчикам быстрый, бесплатный первый проход без зависимостей, который можно запускать локально или в CI.

## Использование

```bash
node accesslint.js ./public
node accesslint.js index.html --ext .html,.htm
```

Завершается с ненулевым кодом выхода, если найдены проблемы, поэтому легко встраивается в CI-пайплайны.
